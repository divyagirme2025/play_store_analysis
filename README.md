# play_store_analysis

## Play Store EDA Analysis

### Objective
To optimize app performance and user satisfaction by analyzing the
Sentiment-to-Rating gap and identifying key triggers like Ads, Bugs, and Pricing.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Key Insights
- Reviews strongly influence installs
- Price has minimal impact on ratings
- Regular updates improve user satisfaction
